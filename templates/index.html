Code Embeddings

Transforms source code into vector embeddings using abstract syntax trees (ASTs) and path-based representations.

Bug Detection

Classifies code snippets as buggy or non-buggy with high accuracy using deep neural networks.

Explainability

Visualizes model attention to understand why code is flagged as potentially buggy.

How It Works

  1. Source code is parsed into Abstract Syntax Trees (ASTs)
  2. ASTs are converted to path-based representations
  3. Neural network generates code embeddings
  4. Model predicts whether code contains bugs
  5. Attention mechanism highlights suspicious code areas
